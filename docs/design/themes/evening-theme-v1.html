Evening — v1

The dark companion to Daylight v1 — same structure, same type, colors only (7a). Charcoal with a whisper of warmth, never black-black. Tokens map 1:1 onto the existing globals.css variables and ship as data-theme="evening". Typography and layout rules are shared: see the Type & Spacing Guide.

Component conventions

Shared with Daylight except where noted — the theme changes values, not rules.

  • Cards — --surface on charcoal, 1px --border, 8px radius, no shadows.
  • Primary button inverts — ivory --foreground fill with dark text (#191918). Secondary stays a tinted --surface-2 fill, no border.
  • Nav header — no bottom border, no active underline; the active item reads by weight + ink alone.
  • Sidebar marker — amber tick hangs in the left margin; row text stays left-justified with its siblings.
  • Voice labels — ✳ coach #6FAE8F · ✎ you #D99A4E · ⇣ imported / ƒ calculated muted; recap sits on --panel when subordinate.
  • Status pills — tinted, never bordered: green/red tints at dark values (#223129 / #362824).
  • Charts — series on chart variants (#3FA873 / #C9604B), grid on --surface-2, axis text muted sans.

Rollout notes

  1. Ships as data-theme="evening" next to daylight; both derive from the same component conventions, so switching is a token swap.
  2. Neutrals are near-gray with only a hint of warmth — if screens ever read brown, the token drifted; correct toward gray.
  3. Contrast check: body #C9C7BF on #191918 ≈ 11:1, muted #93918A ≈ 5.6:1; green #55B183 and red #D97A63 both ≥ 5:1 on ground and surface. Signed figures (+/−, ▲/▼) keep outcomes legible without color.
  4. Replaces the current dark theme's blue-black (#080C12) direction — audit any hardcoded hexes outside globals.css when adopting.
