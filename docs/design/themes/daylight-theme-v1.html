Daylight — v1

The warm light theme settled in 5a / 6a / 6b, packaged for production. Tokens below map 1:1 onto the existing globals.css variables so it can ship as data-theme="daylight". Typography and layout rules live in the Type & Spacing Guide; this document adds the color tokens and component conventions.

In Evening the primary button inverts: ivory ink fill with dark text. Everything else keeps its role.

Component conventions

  • Cards — white on cream, 1px --border, 8px radius, no shadows. Hairlines only inside cards.
  • Controls — no borders: tinted --surface-2 fills for tags/secondary buttons/shell controls; ink-solid --foreground for the primary action and selected pills. 6px radius; full-round for pills/tags.
  • Page header lockup — breadcrumb · 40px gap · date (15px muted) · title 38px with the P&L (19px mono) beside it · one stats pill. No repeated figures below.
  • Voice labels — 13px/600 + glyph: ✳ coach (--coach), ✎ you (--accent), ⇣ imported (muted pill), ƒ calculated (muted). User recap sits on --panel when subordinate.
  • Status pills — tinted, never bordered: Best setup on --green-tint, Worst trade on --red-tint.
  • Charts — --green-chart/--red-chart for series, grid on #F3EFE6, axis text muted 12–13px sans (not mono), entry/exit markers labeled in words.
  • Figures — Geist Mono 600 tabular, always signed, ▲/▼ when standing alone; each figure appears once per view.

Rollout notes

  1. Ship as data-theme="daylight" alongside the existing dark and light themes — no component changes needed for the tokens to take.
  2. The bigger wins are structural and can land incrementally: the header lockup, the stats pill, sentence-case labels replacing mono eyebrows, and borderless controls.
  3. Amber accent replaces blue within Journal/Trades surfaces; audit link and focus-ring styles when switching.
  4. Contrast check: body #4A463E on #FCFAF7 ≈ 8.7:1, muted #8A8375 ≈ 3.6:1 (fine at 13px+ for metadata; don't use for smaller text). Green/red text tokens both ≥ 4.5:1 on white and cream.
